# Data

This folder contains the datasets used in the experimental evaluation of the overlapping phylogenetic tree set completion method.

The evaluation is based on four species groups: **Amphibians, Mammals, Sharks, and Squamates**. The source phylogenetic data were obtained from **VertLife**.

In addition to the prepared datasets, this folder also contains the scripts used to generate and prepare the overlapping input tree sets from the source phylogenies.

## Source

- **VertLife data portal**: [https://vertlife.org/data/](https://vertlife.org/data/)
- **VertLife phylogeny subset tool**: [https://vertlife.org/phylosubsets/](https://vertlife.org/phylosubsets/)
